Module 8 - Assignment 8 (MS Azure)

  • Due No Due Date
  • Points 100
  • Submitting a text entry box or a file upload
 

Using the used cars dataset from Module 5, create a classifier that will predict the price range for a car based on features in the data set. You can consider the following price ranges:

  • Less than $10,000
  • $10,000 - $20,000
  • $20,000 - $30,000
  • Over $30,000

For this exercise setup a free Azure account.

Build the classifier using Azure and exercise the auto-ML features to come up with the at least two classification algorithms.  Refer to the Azure auto-ML tutorial Links to an external site..

Submit the assignment by Day 7 of Module 8. Submit a screen shot of your AutoML results for the assignment.
